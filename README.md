# GiveQuest Website

This is the official website for GiveQuest, a 501(c)(3) nonprofit organization.

## Mission
GiveQuest is committed to directing 100% of donations toward charitable causes, supporting qualified organizations and direct-impact initiatives.

## Purpose of this site
This is a simple, initial version of the website built to:
- Provide transparency and legitimacy
- Support Stripe verification
- Share the mission and structure of GiveQuest

## Tech Stack (initial)
- Next.js (Vercel deployment)

## Future Plans
- Donation integration
- User accounts and referral system
- Campaign tracking and engagement features
